# WorkDays

Calculate the number of business days in a given period.  Also, add convenience methods to Range, Date, and Time.

## Installation

Add this line to your application's Gemfile:

    gem 'work_days'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install work_days
